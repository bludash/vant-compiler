# vant-compiler

Vue single file component compiler.

## Install

#### NPM

```shell
npm i @vant/compiler -S
```

#### YARN

```shell
yarn add @vant/compiler
```

### Options

| Option | Default | Description |
|---|---|---|
| template | `{ preserveWhitespace: false }` | Vue template compiler options |
| babel | `{}` | babel options |
